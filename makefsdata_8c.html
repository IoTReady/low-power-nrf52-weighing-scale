makefsdata: Converts a directory structure for use with the lwIP httpd.

This file is part of the lwIP TCP/IP stack.

Author: Jim Pettinato Simon Goldschmidt

Todo:
:
  • take TCP_MSS, LWIP_TCP_TIMESTAMPS and PAYLOAD_ALIGN_TYPE/PAYLOAD_ALIGNMENT as arguments

Makefsdata can generate all files deflate-compressed (where file size shrinks). Since nearly all browsers support this, this is a good way to reduce ROM size. To compress the files, "miniz.c" must be downloaded seperately.
