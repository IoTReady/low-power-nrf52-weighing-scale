Function adding expander instance.

Should be called for every connected expander. Order of calls define order of pins and ports.

Return values

NRF_ERROR_STORAGE_FULLReturned if trying to add more instances than defined.
otherReturn error code from nrf_twi_sensor nrf_twi_sensor_write
