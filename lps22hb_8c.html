◆ lps22hb_data_read()

ret_code_t lps22hb_data_read ( lps22hb_instance_t *  p_instance,
lps22hb_data_callback_t  user_callback,
lps22hb_data_t *  p_out_data,
uint8_t  samples 
)

Function for reading pressure and temperature data.

Parameters
[in]p_instancePointer to sensor instance.
[in]user_callbackFunction to be called when data is gathered.
[out]p_out_dataPointer to raw data buffer.
[in]samplesNumber of data samples to read.
Note
Data can be read in two ways. With or without sensors FIFO. FIFO mode depends on FIFO mode set using lps22hb_fifo_mode_set function. FIFO is enabled using lps22hb_fifo_enable function. Without FIFO only one sample can be acquired, p_out_data can be pointer to single variable. With FIFO enabled, data can be read in burst mode, p_out_data table has to be same or bigger than number of samples to read.
Returns
Return error code from nrf_twi_sensor nrf_twi_sensor_reg_read
