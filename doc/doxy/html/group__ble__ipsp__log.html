Macros used for creating module logs which can be useful in understanding handling of events or actions on API requests. These are intended for debugging purposes and can be enabled by defining the IOT_BLE_IPSP_CONFIG_LOG_ENABLED to 1.

Note
If NRF_LOG_ENABLED is disabled, having IOT_BLE_IPSP_CONFIG_LOG_ENABLED has no effect.
