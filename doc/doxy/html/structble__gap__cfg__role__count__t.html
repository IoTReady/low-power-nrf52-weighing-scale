Detailed Description

Configuration of maximum concurrent connections in the peripheral role, set with sd_ble_cfg_set.

Configuration of maximum concurrent connections in the different connected roles, set with sd_ble_cfg_set.

Return values
NRF_ERROR_CONN_COUNTThe periph_role_count is too large. The maximum supported sum of concurrent connections is BLE_GAP_ROLE_COUNT_COMBINED_MAX.
NRF_ERROR_RESOURCESThe adv_set_count is too large. The maximum supported advertising handles is BLE_GAP_ADV_SET_COUNT_MAX.
NRF_ERROR_CONN_COUNTThe sum of periph_role_count and central_role_count is too large. The maximum supported sum of concurrent connections is BLE_GAP_ROLE_COUNT_COMBINED_MAX.
NRF_ERROR_INVALID_PARAMcentral_sec_count is larger than central_role_count.
NRF_ERROR_RESOURCESThe adv_set_count is too large. The maximum supported advertising handles is BLE_GAP_ADV_SET_COUNT_MAX.
