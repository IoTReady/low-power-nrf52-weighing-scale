Detailed Description

This is the standardized error codes provided when calling nrf_crypto APIs. The error codes provided here are enumerated based on NRF_ERROR_CRYPTO_ERR_BASE.

Note
Success code, NRF_SUCCESS, is used if the nrf_crypto operation was successful.
